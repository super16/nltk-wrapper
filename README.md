# nltk-wrapper

*nltk-wrapper* is a service over [NLTK](https://www.nltk.org/)
corpora module made with [FastAPI](https://fastapi.tiangolo.com/).

## Development

### Prepare environment

```bash
python3 -m venv env
source env/bin/activate
pip install -r requirements.txt
```

### Run

```
python app/main.py
```
